# Changelog

## [2.0.0] - 2026-01-16

### Added

- Interactive benchmark comparison with selectable datasets
- Multi-language code tabs for C, Python, and Rust
- Live code blocks for quick latency modeling
- Case study datasets with latency/bandwidth charts
- Operations section for deployment, analytics, and compliance

### Changed

- Updated documentation architecture and navigation
- Expanded cross-linking between related topics
- Improved chart theming for light and dark modes

### Fixed

- Broken link validation and sitemap generation
- Footer and navigation consistency across versions

## [1.0.0] - 2025-12-20

### Added

- Core documentation structure and HPC knowledge base
- Mermaid, KaTeX, and Chart.js integrations
- Privacy-first analytics and Giscus comments scaffolding
